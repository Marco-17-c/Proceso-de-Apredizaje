Desarollo Web

HTML,CSS Y JavaScript:

Son los tres pilares fundamentales del desarollo web que trabajan juntos para crear paginas web dinamicas e interactivas.
HTML se encarga de la extructura y el contenido,CSS del diseño y la presentacion y JavaScript de la interactividad y la funcionalidad.

Definiciones

HTML(Hyper Text
Markup Languaje):

Es el lenguaje de marcado que define la estructura y el contenido de una pagina web.Utiliza etiquetas para delimitar diferentes Elementos,como párrafos,encabezados,imagenes,etc

CSS(Cascading Style
Sheets):

Es un lenguaje de estilos que se utiliza para controlar la presentacion de una pagina web.
Permite definir aspectos como los colores,fuentes,tamaños,margenes,etc

JavaScript(Program.
Language):

Es un lenguaje de programacion que permite agregar funcionalidad interactiva a una pagina web.Permite manipular elementos HTML,
responder a eventos del Lorem, ipsum dolor.
